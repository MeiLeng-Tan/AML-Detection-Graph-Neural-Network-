# AML-Detection-Graph-Neural-Network-
AML Detection via Graph Neural Networks (GNN)

This repository presents a Proof of Concept (POC) that leverages Graph Neural Networks (GNNs) to identify suspicious financial activities. By modeling transaction data as a multi-relational graph, this solution captures the structural and behavioral patterns of money laundering—such as "smurfing" and layering—that are often missed by traditional, non-relational machine learning models.
